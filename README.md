# TG Auto Swap - Chrome Extension

TradeGenius 自动兑换工具，支持任意已保存币种之间的自动交替兑换。

本工具为 Chrome 插件，请使用官方版本 Chrome 浏览器运行，无安全隐患！

**当前版本**: v4.0.0

## ✨ 功能特点

- **智能方向判定**：每轮读取源面板实时余额，按 B 优先策略自动选择 A→B 或 B→A
- **单次金额固定**：支持设置每轮 A→B 的固定金额（推荐稳定币作为起始）
- **停止自动回到 A**：停止时若 B 有残留，自动补跑 B→A（含失败重试，最多 3 次）
- **交易设置保护**：一键获取网站滑点/聚合器设置为模板，启动前自动校准
- **定时启动**：每日固定时刻自动启动循环（基于 chrome.alarms）
- **Gas 监测**：支持 10+ 条链实时 Gas 价格显示
- **多链币种支持**：严格按 chain badge/菜单匹配目标链，绝不选错
- **调试模式**：执行到「确认」按钮前停下，不消耗 Gas（用于测试流程）
- **授权系统**：激活码保护，试用模式可体验

## 🚀 使用步骤

### 1. 配置币种

1. 打开 `https://www.tradegenius.com/zh/trade`（**必须切换为中文**）
2. 在网站上将要兑换的两个币种收藏到「已保存」
   - 第一个（上方）= **A 起始**：想保留/回到的币种
   - 第二个（下方）= **B 目标**：会被消耗的币种
3. 点击扩展图标 → 「设置」标签页 → 「获取币种」
4. 检查币种卡：A/B 徽章颜色正确（绿色=A，灰色=B），可点击「设为 A」交换

### 2. 设置交易参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| **单次交易金额** | A→B 每轮使用的 A 币种数量（0 = 全部 MAX）| 0 |
| **最大次数** | 执行次数上限，0 = 无限 | 0 |
| **多链目标链** | 币种为多链时，需从下拉菜单**手动选择**要交易的链 | — |

### 3. 设置执行参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 最大 Gas | Gas 超过此值时等待，0 = 不限 | 0 |
| 最大滑点 | 滑点超过此值时刷新重试（百分比），0 = 不限 | 0.31% |
| 执行间隔 | 每次兑换之间的等待时间 | 5 秒 |
| Gas 刷新间隔 | Gas 价格刷新频率 | 10 秒 |
| **最小兑换价值** | 源币余额低于此值不发起兑换（USD）| 10 |
| **最小残留价值** | 停止时 B 低于此值视为已回到 A（USD）| 0.5 |
| **调试模式** | 执行到确认按钮前停下，不真实兑换 | 关闭 |

**滑点推荐**：稳定币 ≥ 0.05%，其他币种 ≥ 0.3%。

### 4. 交易设置保护（推荐）

为防止网站设置被误改影响交易：

1. 在 TradeGenius 交易设置里调好理想配置（滑点、聚合器、链等）
2. 回到插件 → 「设置」→ 交易设置保护 → **获取当前设置为模板**
3. 勾选「启用保护」
4. 每次启动循环前，插件会自动校准网站设置到模板状态

### 5. 定时启动

- 切到「定时」标签页
- 勾选「启用定时启动」+ 设置时间（24h 格式）
- 每天此时刻自动启动循环
- **浏览器必须保持打开**才会触发

### 6. 开始运行

1. 切到「兑换」标签页
2. 确认状态大卡显示「就绪」（绿色圆点）
3. 点击「开始」
4. 状态变「运行中」（橙色脉动圆点）

## 📊 交易统计

- **粉色数字**：本轮交易量（扩展重载后重置）
- **黄色数字**：历史累计交易量（持久保存）
- **×按钮**：清除历史记录

> 统计数据仅供参考，以实际交易为准。

## 🎨 状态说明

| 状态 | 含义 |
|------|------|
| **就绪** | 已连接 TradeGenius 页面，可以启动 |
| **运行中** | 正在执行自动兑换 |
| **正在回到 A...** | 停止后的补跑流程 |
| **已停止（已回到 A）** | 补跑完成，手里是 A |
| **已停止（B 残留 $X）** | B 仍有残留，可手动处理 |
| **已停止（补跑失败）** | 滑点过高等原因链上失败 |
| **未连接** | 请打开 TradeGenius 交易页并切换为中文 |
| **授权无效/已过期** | 需要重新激活授权码 |

## 🔒 多链币种安全保障

所有多链币种兑换都有严格校验：

- **源面板**：按 chain badge 图标精确匹配链版本
- **目标面板**：按 hover 菜单文本规范化后匹配
- **找不到目标链 → 立即停止并报错**，绝不选错链

## ⚠️ 注意事项

- 运行时请保持 TradeGenius 页面在前台
- 建议首次使用开启**调试模式**测试流程（不消耗 Gas）
- 合理设置 Gas 和滑点阈值
- A 币种建议使用稳定币（USDT/USDC），并配合「单次交易金额」固定每轮金额

## ❓ 常见问题

**Q: 显示「未连接」怎么办？**
A: 刷新 TradeGenius 页面，确保在 `/trade` 页面，且语言切换为中文。

**Q: 授权码激活失败？**
A: 检查授权码是否正确，或尝试点击「验证」按钮刷新状态。

**Q: 兑换一直失败？**
A:
- 检查钱包余额和 Gas 费是否充足
- 调大滑点阈值
- 检查交易设置保护模板的滑点值是否合理
- 尝试在网站手动兑换一次确认能否成功

**Q: 补跑失败怎么办？**
A: 补跑已内置 3 次重试。若多次失败（一般是滑点保护触发），手动去网站清理 B 残留即可。

**Q: 定时启动没触发？**
A: 浏览器必须保持打开状态，且 TradeGenius 页面已连接。关闭浏览器期间的触发不会补跑。
