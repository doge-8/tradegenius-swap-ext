# TG Auto Swap - Chrome Extension

TradeGenius 自动兑换工具，支持任意已保存币种之间的自动交替兑换。

本工具为chrome插件，请使用官方版本chrome浏览器运行，无安全隐患！

**当前版本**: v3.0

## 功能特点

- 支持任意两个已保存币种之间自动交替兑换
- 支持多链币种（自动选择目标链）
- Gas 价格监测（支持 10+ 条链）
- 滑点检查与自动重试（基于 USD 价值计算）
- 交易量统计（本轮/历史累计）
- 转盘自动旋转（开发中）

## 安装

1. 解压 `tg-swap-ext.zip`
2. 打开 Chrome，进入 `chrome://extensions/`
3. 右上角开启「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择解压后的文件夹

## 使用步骤

### 1. 配置币种

1. 打开 https://www.tradegenius.com/zh/trade
2. 在网站上将要兑换的两个币种使用⭐️收藏添加到「已保存」
3. 点击扩展图标，进入「设置」标签页
4. 点击「刷新币种」获取已保存的币种
5. 确认显示的两个币种正确
6. 切换到中文模式

### 2. 设置参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 最大次数 | 执行次数上限，0 = 无限 | 0 |
| 最大 Gas | Gas 超过此值时等待，0 = 不限 | 0 |
| 最大滑点 | 滑点超过此值时刷新重试，0 = 不限（单位：百分比） | 0.31% |
| 执行间隔 | 每次兑换之间的等待时间 | 5 秒 |
| Gas 刷新间隔 | Gas 价格刷新频率 | 10 秒 |

**滑点推荐设置**：稳定币 ≥ 0.05%，其他币种 ≥ 0.3%

### 3. 开始运行

1. 切换到「兑换」标签页
2. 确认连接状态为「已连接」
3. 点击「开始」开始自动兑换
4. 点击「停止」停止运行

## 交易统计

- **绿色数字**：本轮交易量（扩展重载后重置）
- **黄色数字**：历史累计交易量（持久保存）
- **×按钮**：清除历史记录

> 注：统计数据仅供参考，以实际交易为准

## 状态说明

- **已连接** - 扩展已与 TradeGenius 页面建立连接
- **运行中** - 正在执行自动兑换
- **已停止** - 已手动停止或达到执行上限
- **授权无效/已过期** - 需要重新激活授权码

## 多链币种

如果配置的币种支持多链：
- 币种名称会显示为橙色高亮
- 在设置页面选择「多链目标链」
- 兑换时会自动选择指定的链

## 注意事项

- 运行时请保持 TradeGenius 页面在前台
- 每次兑换会使用全部余额，如需保留部分资金请先转出
- 首次使用建议先观察几次兑换确保正常
- 建议根据实际情况设置合理的 Gas 和滑点阈值

## 常见问题

**Q: 显示「未连接」怎么办？**
A: 刷新 TradeGenius 页面，确保在 trade 页面。

**Q: 授权码激活失败？**
A: 检查授权码是否正确，或尝试点击「验证」按钮刷新状态。

**Q: 兑换一直失败？**
A: 检查钱包余额和 Gas 费是否充足，尝试调大滑点阈值。
